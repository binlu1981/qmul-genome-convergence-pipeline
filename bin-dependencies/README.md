# Dependencies

This folder contains third-party dependencies required by this package (This also folder contains the 'runCmd.pl' perl wrapper script for these):

 - runCmd.pl
 
 - NewickUtils (v?)
     - nw_prune
     - nw_display
     
 - PAML (v?)
     - codeml
     - baseml
     - evolver
     - 'dat' folder containing matrices for substitution models
    
 - RAxML (v7.2.8)
     - raxmlHPC
     - raxmlHPC-PTHREADS
